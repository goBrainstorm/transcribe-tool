# Models
## This section is where the transcribe models belong.

### How to use models?
You can download them with the provided tool from the [ggml-org/whisper.cpp](https://github.com/ggml-org/whisper.cpp/tree/master/models) repo. 

#### The easiest way is to download them with their provided [script](https://github.com/ggml-org/whisper.cpp/blob/master/models/download-ggml-model.sh). It should be also in this folder (no guarantee for always being up to date).
