# Models
## This section is where the transcribe models belong.

### How to use models?
You can download them with the provided tool from the [ggml-org/whisper.cpp](https://github.com/ggml-org/whisper.cpp/tree/master/models) repo. 

#### The easiest way is to download them with their provided [script](https://github.com/ggml-org/whisper.cpp/blob/master/models/download-ggml-model.sh).
This script can be found in this repo as a submodule under "whisper.cpp/models/download-ggml-model.sh"
